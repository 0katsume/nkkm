# NKKM
This is an SMS bomber created by the coder Katsume. Only for Ukraine.

# INSTALLING
For Linux
```
git clone https://github.com/0katsume/nkkm
pip install -r requirements.txt
python nkkm.py
```
For Termux
```
pkg install git python python3
git clone https://github.com/0katsume/nkkm
pip install -r requirements.txt
python nkkm.py
```
